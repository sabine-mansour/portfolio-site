Chat App

Chat app is a chat mobile application built using React Native. The app provides users with a chat interface and options to share images and their location.

Mobile Application | Google Firestore Database React Native | Expo | GiftedChat

Key Features

The Chat app provide users with:
  • A page where they can enter their name and choose a background color for the chat screen before joining the chat.
  • A page displaying the conversation, as well as an input field and submit button.
  • The chat must provide users with two additional communication features: sending images and location data.
  • Data which get stored online and offline.

Objectives

The focus of this project was to create a mobile application using React Native, Expo, and Google Firestore Database. The Chat App authenticates users anonymously via Google Firebase authentication. Users can then exchange messages, images, and their location.
